### 5 建筑工程的造价
1. 背景: 建筑工程的造价，需要根据工程量来计算。不同的工程，工程量的计算方式不同: 有的按面积，有的按体积，有的按高度等。
2. 要求: 编写 OO 程序，实现灵活的工程造价的计算。
注意: 面积，体积  不用实际计算，作为参数直接给出。

### 6 计算人均面积
1. 背景: 一个家庭分得若干土地，土地的形状不一，请写一个程序计算人均面积。
2. 要求: 编写 OO 程序完成，要注意接口和集合（ArrayList）在本次作业中的应用。

### 7 基于Map的中英文词典
本次作业：基于Map的中英文词典
要求：
一、基本要求（必做）
1.单词词条（英文、中文、例句）的增加、修改、删除；
2.单词查询：用户输入英文单词，程序给出中文含义、例句；
3.词典遍历：显示所有单词
二、扩展要求（选做）
1.按字母顺序遍历整个词典；
2.随机测验：随机抽取若干单词，用户给出答案，程序判断对错，并统计正确率。
